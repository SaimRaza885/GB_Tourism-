# 🏔️ Gilgit Tourism — Explore the Beauty of Gilgit-Baltistan

A full-featured tourism web application for **Gilgit-Baltistan, Pakistan** — one of the world's most breathtaking mountain regions. Built with **Next.js**, **React**, and **Tailwind CSS**, this platform helps travelers discover tours, hotels, car rentals, and travel stories from the Karakoram.

🔗 **Live Demo:** [gb-tourism-smoky.vercel.app](https://gb-tourism-smoky.vercel.app)

---

## ✨ Features

- 🗺️ **Tour Packages** — Browse curated all-inclusive tour packages with pricing, duration, and ratings
- 🏨 **Hotel Listings** — Explore premium accommodations across Hunza, Skardu, Gilgit, and more
- 🚗 **Car Rentals** — Book reliable 4x4 vehicles with professional drivers
- 📸 **Photography Services** — Dedicated photography service page for travelers
- 📖 **Travel Blog** — Real travel stories and destination guides
- 🌍 **Region Explorer** — Browse major regions: Hunza Valley, Skardu, Ghizer, Astore & Rama
- 📬 **Contact & Booking Form** — Integrated with EmailJS for real email delivery
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop
- ⚡ **Optimized Performance** — Next.js Image optimization and dynamic routing

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/SaimRaza885/GB_Tourism-.git

# Navigate to the project
cd GB_Tourism-

# Install dependencies
npm install
# or
pnpm install

# Run the development server
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm run start
```
